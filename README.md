# ЧПУ Шпаргалка

> Нажал кнопку — и в носу поковырял. Интерактивная шпаргалка для операторов ЧПУ-станков.

**[→ Открыть приложение](https://babadjii.github.io/cnc-cheatsheet/)**

![ЧПУ Шпаргалка](icon-192.png)

---

## О проекте

Веб-приложение-шпаргалка по трём основным стойкам ЧПУ: **Fanuc**, **Siemens** и **Heidenhain**. Два режима — для тех кто только начинает, и для тех кто уже в теме.

Работает как **PWA** — устанавливается на смартфон как обычное приложение, работает офлайн.

---

## Режимы

| Режим | Для кого |
|-------|----------|
| **Новичок** | Базовые G/M-коды, смена инструмента, привязка нуля, мерительный инструмент, чтение чертежей, сборка инструмента |
| **Продвинутый** | Циклы, смещения, коррекция инструмента, 5-осевые операции, примеры программ, генератор программ |

Чеклист запуска и FAQ доступны в обоих режимах.

---

## Разделы

### Новичок
- **G-коды (база)** — основные коды позиционирования и движения
- **M-коды (база)** — управление шпинделем, СОЖ, останов
- **Смена инструмента** — команды и особенности на каждой стойке
- **Сборка инструмента** — типы патронов, цанг, оправок; советы по выбору
- **Мерительный инструмент** — штангенциркуль, микрометр, нутромер; как не облажаться
- **Чтение чертежей** — виды, разрезы, допуски, посадки, шероховатость
- **Привязка нуля детали** — ручная привязка по G54 без циклов

### Продвинутый
- **G-коды** — расширенный список с описанием
- **M-коды** — полный список
- **Циклы** — сверление, резьба, карман, контур
- **Смещения и нули** — работа с системами координат
- **Коррекция инструмента** — длина, радиус, износ
- **Работа с нулями** — G54–G59, расширенные смещения
- **Опасные различия** — где Fanuc, Siemens и Heidenhain ведут себя по-разному
- **Примеры программ** — реальные программы с комментариями
- **5-осевые операции** — TCPC, RTCP, наклон плоскости
- **Генератор программ** — конструктор типовых циклов

### Всегда доступно
- **Чеклист запуска** — 5 фаз, 27 пунктов, не забудешь ничего
- **FAQ** — 35 вопросов по 4 категориям: нули, инструмент, программы, аварии

## Стойки

| | Fanuc | Siemens | Heidenhain |
|---|---|---|---|
| Цвет | 🔵 синий | 🟠 оранжевый | 🟢 зелёный |
| Нули | G54–G59 | Zero Offset | Preset / Datum |
| Циклы | G73–G89 | CYCLE8x | Циклы 2xx |
| Редактор | MDI / EDIT | MDA / JOG | MDI / PGM |

## Технологии

- Чистый HTML / CSS / JS — никаких фреймворков
- PWA: `manifest.json` + Service Worker (офлайн-кеш)
- Темная тема only — `data-theme="dark"` зашит намертво
- Адаптив: десктоп (боковое меню) + мобайл (нижний дровер)
- Таблицы на мобиле → карточки с `data-label`

## Установка как приложение

**Android (Chrome):**
1. Открыть [babadjii.github.io/cnc-cheatsheet](https://babadjii.github.io/cnc-cheatsheet/)
2. Меню → «Добавить на главный экран»

## Разработка

```bash
# Клонировать
git clone https://github.com/BaBaDjii/cnc-cheatsheet.git
cd cnc-cheatsheet

# Открыть локально — просто открой index.html в браузере
# Или поднять dev-сервер:
python3 -m http.server 8080
```

Файлы:
```
index.html   — всё содержимое (~2800 строк)
style.css    — стили (~2000 строк)
app.js       — логика, генератор, чеклист (~1000 строк)
sw.js        — service worker
manifest.json
icon-192.png
icon-512.png
```
